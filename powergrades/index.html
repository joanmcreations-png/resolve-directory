Frequently asked

What is a PowerGrade?

A PowerGrade is a saved DaVinci Resolve node tree — the complete grade including every node, LUT and adjustment — stored as a .drx file you can reuse on any project.

How do I import a PowerGrade?

On the Color page, open the Gallery panel, create or pick a PowerGrade album, then drag the .drx file into it. Right-click a clip and choose Apply Grade.

Do PowerGrades need Resolve Studio?

No. PowerGrades work in the free version of DaVinci Resolve as well as Studio.
