How to install this in DaVinci Resolve

Open the Color page in DaVinci Resolve, then in the Gallery panel create or select a PowerGrade Album. Drag the downloaded .drx file into that album. To use it, right-click any clip's node graph and choose "Apply Grade", or drag the PowerGrade thumbnail directly onto a node.
